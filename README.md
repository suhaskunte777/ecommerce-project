# Product API Documentation
## 1.0.0

### Setup

```bash
git clone 
```

```bash
composer install
```

```bash
cp .env.example .env
```

```bash
php artisan key:generate
```

Setup the database and run migrations with seeders

```ENV
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=web_server
DB_USERNAME=root
DB_PASSWORD=
```

```bash
php artisan migrate --seed
```

```bash
npm install
```

```bash
php artisan serve
```
